Hra: sestav rezervační systém v Pythonu a SQLite

Student má za úkol přetahat bloky Python kódu do správného pořadí tak, aby vznikl funkční konzolový rezervační systém. Jakmile je pořadí správně, dole se automaticky vygeneruje celý finální skript ke zkopírování.

Python SQLite drag & drop nápovědy výstupní kód ke zkopírování

Jak se to hraje

  • Vlevo jsou pomíchané bloky kódu.
  • Vpravo jsou sloty s nápovědami, kam jednotlivé bloky patří.
  • Blok přetáhni do slotu. Když chceš, můžeš ho zase vrátit zpět do levého panelu.
  • Tlačítko Zkontrolovat ověří správné pořadí.
  • Po správném složení se dole objeví celý hotový Python skript.

Skóre a průběh

Tohle je schválně udělané jako malá logická hra. Čím méně kontrol a nápověd použiješ, tím lepší skóre získáš.
